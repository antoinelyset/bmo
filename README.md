# BMO (Beemo)

<img src="https://raw.github.com/antoinelyset/bmo/master/bmo.png"
     alt="BMO"
     align="right" />


BMO is gem to Push Notifications to iOS (via APNS) and Android (via GCM)

## Installation

```
gem install bmo
```

In Gemfile :

```
gem 'bmo'
```
